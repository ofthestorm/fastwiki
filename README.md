# fastwiki

A CLI tool for Wikipedia query based on npm. Use [MediaWiki](https://www.mediawiki.org/wiki/MediaWiki) as data source.

## How to install

```bash
$ npm install -g fastwiki
```

## How to use 

```bash
$ wiki <keyword>
```
